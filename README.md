# VisioBoard 🖌️📌

VisioBoard ek **Next.js** aur **React with TypeScript** ka istemal karne wala **modern productivity tool** hai jo users ko **visual notes aur ideas** ko organize karne me madad deta hai. 🚀

## 🌟 Features

✅ Drag & Drop Sticky Notes 📌  
✅ Interactive & Resizable Whiteboard 🎨  
✅ Multi-User Collaboration 🤝  
✅ Fast & Optimized Performance ⚡  
✅ Dark/Light Mode Support 🌗
